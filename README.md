# フロントエンド(Typescript-nuxt-api 商品登録アプリ)

## 使用した技術要素

| 技術 | バージョン |
|:-----------|------------:|
| Nuxt       |         @nuxt/cli v2.13.3 | 
| Vuex     |      3.5.1 |   
| TypeScript       |         0.4.10 |        
| node.js       |            v14.15.5 |  
| bootstrap       |            4.5.0 |  


### 環境構築手順
# node.jsをインストール
$ brew install node  

# yarnをインストール
$ npm install -g yarn
$ yarn install

# サービスの起動
$ yarn run dev
* HomePage URL : http://localhost:5000


# java-apiを起動し、連携する
* java-apiのapplication.ymlで
frontend.url: http://localhost:5000に変更
